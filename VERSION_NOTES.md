# v2 변경사항

- 메인 메뉴를 오늘의 안전 현황, 5×7 시간표, 교내 위험지도, 위험 분석으로 단순화
- 학교·학급 등록과 지도 좌표 관리를 관리자 설정 안으로 이동
- 월~금 × 1·2·3·4·점심·5·6교시 고정 시간표 구현
- 이동수업 셀에 ↗ 표시
- 시간표 셀 선택 후 이동층, 장소, 이동경로, 활동/비고 수정
- 학교 전체 학급의 이동수업을 시간대별로 합산
- 복도·계단별 예상 이동 인원 및 수용기준 대비 혼잡도 계산
- 학교 지도 위 노랑·주황·빨강 위험 원 표시
- 상행·하행 동선 중첩, 저학년 비중, 계단 여부를 위험도에 반영
- 위험구간별 교사 0~2명 배치 권고
- 6개 학급이 포함된 예시 데이터 자동 생성 기능 추가
