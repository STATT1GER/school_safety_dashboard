# 학교 시간표 기반 교내 이동위험 Streamlit 대시보드 v2

학교 전체 학급의 시간표와 이동수업 경로를 등록한 뒤, 특정 교시 시작 전 복도·계단의 예상 이동 인원을 합산하여 학교 지도 위에 위험도를 표시하는 MVP입니다.

## 이번 버전의 핵심 화면

### 1. 오늘의 안전 현황
- 등록 학급 수
- 이동 학급 수와 예상 이동 학생 수
- 시간대별 학교 위험지도
- 위험구간 Top 3
- 교사 배치 권고

### 2. 5×7 학급 시간표
- 월·화·수·목·금 5열
- 1·2·3·4·점심·5·6교시 7행
- 셀에 과목 표시
- 이동수업이면 `↗` 표시
- 셀 선택 후 이동층, 도착 장소, 이용 복도·계단, 활동/비고 수정
- 시간표 사진 첨부 및 CSV·Excel 불러오기

### 3. 교내 위험지도
- 학급별 이동경로에 포함된 복도·계단별 예상 이동 인원 합산
- 노란색: 주의
- 주황색: 혼잡
- 빨간색: 집중 관리
- 상행·하행 동선 중첩과 저학년 비중 반영
- 위험구간별 교사 배치 인원 권고

### 4. 위험 분석
- 요일별·교시별 최고 위험점수
- 가장 위험한 시간대와 구간
- 관련 이동 학급과 운영 권고

학교·학급 등록, 학교 지도, 복도·계단 좌표, 데이터 관리는 사이드바의 접힌 `관리자 설정` 안에 배치했습니다.

## 설치해야 하는 프로그램

### 이미 Python과 uv가 설치된 경우

추가 프로그램 설치는 필요하지 않습니다. ZIP 압축을 푼 폴더에서 아래 명령을 실행합니다.

```powershell
uv sync
uv run streamlit run app.py
```

### uv가 없고 Python만 있는 경우

Python 3.11 이상이 필요합니다.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

브라우저가 자동으로 열리지 않으면 다음 주소로 접속합니다.

```text
http://localhost:8501
```

패키지를 한 번 설치한 뒤에는 `run_windows.bat`를 더블클릭해 실행할 수 있습니다.

## 최초 실행 순서

앱을 처음 확인할 때는 다음 순서가 가장 간단합니다.

1. 사이드바에서 `관리자 설정` 펼치기
2. `설정 화면 열기` 체크
3. `데이터 관리` 선택
4. `예시 데이터 생성` 클릭
5. 설정 체크를 해제하고 `오늘의 안전 현황`으로 이동
6. `안전초등학교`, `화요일`, `3교시 시작 전` 선택

예시 데이터에는 6개 학급과 이동수업 경로가 들어 있어 위험지도 기능을 즉시 확인할 수 있습니다.

실제 학교 데이터를 등록할 때는 다음 순서로 진행합니다.

1. 관리자 설정 → 학교·학급에서 모든 학급 등록
2. 관리자 설정 → 학교 지도·이동구역에서 지도와 복도·계단 등록
3. 각 학급의 5×7 시간표 입력
4. 이동수업 셀에서 이동경로 지정
5. 위험지도와 위험분석 확인

## 위험도 계산 방식

현재 위험도는 실제 사고발생 확률이 아니라, 다음 요소를 조합한 상대적 이동위험지수입니다.

- 해당 구역을 이용하는 예상 학생 수
- 구역별 수용기준 대비 혼잡비율
- 계단 여부
- 상행·하행 동선 중첩
- 저학년 이동 비중

교사 배치 권고는 위험점수와 수용기준 초과 정도를 기준으로 0명, 1명, 2명으로 제시합니다.

## 학교 지도 설정 방법

학교 지도는 PNG, JPG, JPEG, WEBP 파일을 사용할 수 있습니다.

각 구역에 다음 값을 입력합니다.

- 구역명: `서쪽계단`, `2층 중앙복도`처럼 시간표 이동경로에서 사용할 이름
- 층
- 구역유형
- 수용기준: 한 번의 교시 전환 시간에 비교적 안전하게 통과할 수 있다고 정한 인원
- 지도 X(%): 왼쪽 0, 오른쪽 100
- 지도 Y(%): 위 0, 아래 100
- 표시반경(%): 지도 위 위험 원의 크기

`현재 좌표 미리보기`를 이용해 위치를 조정할 수 있습니다.

## 시간표 CSV·Excel 형식

권장 열은 다음과 같습니다.

```text
요일,교시,과목,이동수업,도착장소,이동층,이동경로,활동/비고
```

교시는 다음 값 중 하나를 사용합니다.

```text
1교시,2교시,3교시,4교시,점심,5교시,6교시
```

이동경로는 등록된 구역명을 다음처럼 입력합니다.

```text
3층 중앙복도 → 서쪽계단 → 강당 앞
```

## 현재 포함하지 않은 기능

- 시간표 사진 OCR 자동 인식
- 실제 사고 데이터 기반 사고확률 예측
- 사고 장소 → 활동 → 사고형태의 Sankey 경로 분석
- 교사 근무표와 연동한 실제 배치 자동 확정
- 사용자 로그인과 학교별 서버 데이터베이스

현재 버전은 화면과 데이터 구조를 검증하기 위한 로컬 MVP입니다.
